Website Features

Web Development

This website was designed for Project Milestone #1 and is the start to three iterations. This is iteration one and includes a Home Page, an About Page, and a Contact Page. The website also includes the start to a functional Login/Signup feature. More to come with furture iterations.

Design

The Design for this website was to keep it simple and clean. Colors are kept simple to avoid any weird mix of colors. Keeping a darker scheme of colors makes it easy on the eyes of the user.

Form Validation

A Login/Signup button has been created to allow the user to Login or Signup. In this stage of the website, an alert notifes user of a successful login or signup. Form validation has also been used to ensure valid emails are used. Passwords must also be the same for signup and meet a 8 character minimum.
